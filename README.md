This is a basic web-based calculator built with HTML, CSS, and JavaScript.
It supports basic arithmetic operations: addition, subtraction, multiplication, and division.

The calculator is designed to be simple, beginner-friendly, and easy to understand. Each button has its own ID, and the JavaScript is written in a straightforward way, without loops or advanced functions.

Features

Numbers: 0–9

Operators: +, −, ×, ÷

Decimal point .

Clear (C) button

Equals (=) button calculates the result

Simple CSS for a modern dark theme

Clear button briefly changes color when pressed

How to Use

Open index.html in your browser.

Click on the number buttons to enter digits.

Click on an operator button to perform calculations.

Click = to see the result.

Click C to clear the display.

File Structure
calculator/
├── index.html      # Main HTML file
├── styles.css      # CSS for styling
└── script.js       # JavaScript logic
Notes

The calculator uses eval() in JavaScript for simplicity.

Operators × and ÷ are converted to * and / internally for computation.

This project is ideal for beginners learning how to manipulate the DOM and handle button events.
